# Security Policy

## Supported Versions

Currently, only the latest version of this project is actively supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please DO NOT open a public issue.

Instead:

* Contact me directly!

## What to Expect

* You will receive acknowledgment within a few days
* We will investigate and aim to resolve the issue promptly
* If valid, a fix will be released and credited where appropriate

## Scope

This applies to:

* Test framework behavior
* Dependency vulnerabilities
* Any misuse that could affect users of this project
